# Расширение ModX Evolution 2.x Commerce сервиса «Экспресс Платежи: ЕРИП»
Расширение CMS ModX для интеграции с сервисом «Экспресс Платежи». Расширение позволяет выставить счет в системе ЕРИП, получить и обработать уведомление о платеже в системе ЕРИП.
